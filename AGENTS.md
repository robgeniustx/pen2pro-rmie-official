# PEN2PRO AGENTS.md

When working in this repo:

- Treat the product as a business planning platform for everyday founders.
- The app must generate structured outputs, not long freeform essays.
- All roadmap outputs must support:
  - business_snapshot
  - startup_requirements
  - licenses_and_compliance
  - tools_and_software
  - pricing_strategy
  - launch_plan_30_days
  - operations_plan_90_days
  - scale_plan_12_months
  - risk_flags
  - sources
- Prefer incremental pull requests.
- Preserve existing deployment behavior unless explicitly changing it.
- Ask for missing schema details in code comments or TODOs, not by deleting structure.
